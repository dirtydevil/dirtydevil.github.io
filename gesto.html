Gesto-Paint as name explains, Painting using hand gestures. I got inspiration to do this project after watching Pranav Mistry's TED Talk on “Sixth-Sense”. Putting a colored cap on fingers, doing gestures, computer recognizing and acting accordingly seemed super cool. After being fascinated by movie “Minority Report” and then Sixth Sense, I thought of this idea just to kickstart my passion of computer vision and HCI.

Idea was “ While Talking about Interaction with Computer, we need to talk in its language i.e machine language. Now the question arises why do we need to learn computer's language when we are using it for our ease. As all know, Gestures are the best part in expressing one. Then We can use our hand gestures to make computer work accordingly. So gesto-Paint is a hand gesture based project that enhance user interaction with computer.


I used OpenCV to recognize Red Colored marker and pymouse to control cursor, though I didn't created a paint application but my prototype's working was showing that it will paint too. Later on, I tried this same project with OpenFramework too.


To know more about this project, you can check its GitHub Repo.
