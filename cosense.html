CoSense: Creating Shared Emotional Experiences


CoSense is a prototype wearable interface that shares a user’s first person video and their current emotional state with a remote user in order to create a shared emotional experience.



Prototype included a combination of sensors like GSR, Blood Oxymeter and Hear rate sensor to recognise emotions. Google glass was used to capture local user’s POV and send to remote user (laptop for prototype) using spydroid -ipcamera.

My task was to create an interface which could be used to visualise the emotional states. After reading few research papers, I found colour could be a way to visualise emotion.

A user evaluation was conducted to explore which interface cues best helped a remote user to understand what the local user was feeling.

We evaluated from 4 different interfaces, which were

  1. Just video sharing,

  2. Video along with raw data from Blood oxygen level, heart rate, skin conductance, skin resistance and skin voltage in form of graphs,

  3. Video with colour overlay based on particular emotions, and

  4. Coloured overlay video along with all the raw data graphs as mentioned in 2nd interface.

The result showed simple visual cues (3rd interface) provided a significantly enhanced experience over no cues at all ( interface 1), or a more detailed data representation (interfaces 2 and 4).

Our short paper about the research findings from this project is being published in “Work in Progress” category at CHI 2015 Seoul.
